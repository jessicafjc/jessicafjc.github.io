前置环境

  1. 代码编译器我选择Hbulider X 官网下载
  2. 在Hbuilder新建一个空白项目test_demo，包含index.html、css文件夹、js文件夹、img文件夹
  3. 前往Bmob官网查看文档并下载对应SDKGitHub源码地址
  4. 在Bmob的控制台云数据库新建项目、新建表、设计数据结构并导入数据

在项目中接入Bmob云数据库

  1. 引入bmob.js
    将sdk中的dist/Bmob-2.2.1.min.js导入项目的js文件夹下
    在index.html中引入 <script src="js/dist/Bmob-2.2.1.min.js"></script>
    新建一个js文件main.js
  2. 连接对应的bmob数据库
    Bmob.initialize("你的Secret Key", "你的API 安全码");

    获取_User列表内所有数据，因为_User是系统内置用户表，出于隐私安全考虑，密码不可见，且无法用明文方式进行参数传递。
