iOS 开发基础实践

发表于 2023-06-12 更新于 2023-06-29 分类于 教学研讨 阅读次数： Valine：
上海商学院-大二-计算机类产学合作实践

课程信息

  • 总时长：18小时 27课时

  • 授课时间：2023-06-19 至 2023-06-30 周一至周五 09：00 - 11：00

  • 授课形式：线下授课 图文楼319机房

  • 软硬件：Mac OS

  • 课程成果：实训报告 + 实训成果zip，命名为 学号+姓名 ，zip包含：

    • 工程文件，包含课程随堂代码，个人或小组开发的作品代码
    • 个人或小组作品设计说明（展现形式不限，pdf或PPT或keynote或视频）

  • 结课要求：出勤40分+课程成果60分

    • 出勤按每日签到表核算
    • 课程成果由任课老师打分

  • 学生提交要求：现场递交或邮件递交

    • 每位学生将递交两项内容，分别为 实训报告.doc 和 实训成果.zip
    • 命名为 学号+姓名
    • 现场递交截止日期：2023年6月30日 11:00
    • 邮件递交截止日期：2023年7月2日 11:00
    • 递交结果查询：SBS物联网实训课-iOS 开发基础实践-结课递交表

  • 网盘链接: 链接: https://pan.baidu.com/s/1jAIdgRJOFjSSJN36LOTy7A 提取码: ha2b

  • 主讲： fujiachen@qingmaoedu.com

  • 交流论坛： Create4World学生论坛

课程大纲

  1. App 应用开发——iOS 开发概览
    1. 软件的生命周期
    2. 产品策划思维
    3. iOS开发前置条件
  2. App 应用开发——Swift基础语法
    1. 面向对象的编程
    2. Swift 基础语法
  3. App 应用开发——Swift基础语法
    1. 循环语句
    2. 条件判断
    3. 结构体、类、枚举
  4. App 应用开发——基础UI组件学习
    1. UI 布局
    2. 基础控件：Image、Text
    3. 界面的模块化开发、MapKit
    4. 搭建基础单页面
  5. App 应用开发——基础UI组件学习
    1. 轮播图
    2. 横向滚动视图
    3. 纵向滚动视图
    4. 简单资讯页面开发
  6. App 应用开发——TabView与JSON数据传输
    1. 导航栏设置与页面跳转
    2. 网络数据获取：JSON数据
    3. 实现一个实时天气信息显示页面
    4. TabView页面结构开发
  7. App 应用开发——多数据获取与WebKit
    1. JSON获取多条数据信息
    2. Webkit的引用
    3. 实现新闻列表页及详情页开发
    4. 实现简单网格化视图
    5. 完善TabView的多个页面开发
  8. iOS项目开发实践
    1. 结课项目开发指导
    2. 技术答疑
  9. iOS项目开发实践
    1. 项目开发指导
    2. 技术答疑
