不定时更新一些用过的插件

文章加密

安装npm install --save hexo-blog-encrypt
将 “password” 字段添加到文章头

纯数字不好使

参考文章

Hexo博客技巧：对文章进行加密

插入图片

安装npm install https://github.com/CodeFalling/hexo-asset-image –save

注意这一步一定要使用上面的安装方法，使用官网的代码则会出现保存文件为.com的现象

hexo 添加live2d看板动画

  1. 安装npm install --save hexo-helper-live2d
  2. 下载各种动画model
  3. 配置博客站点配置文件，**_config.yml**

    参考文章

    hexo 添加live2d看板动画

首页不显示全文，显示阅读更多

  1. 打开博客主题配置文件 themes/next/_config.yml，修改excerpt_description: 为true，好像现在版本的next主题默认是true
  2. 两种方式
    1. 在需要截断的地方输入 <!--more-->
    2. 在md开头写摘要 description: 这是显示在首页的概述，正文内容均会被隐藏。
  3. 重启部署即可 hexo clean hexo d -g

添加统计数据

百度统计

  1. 登录百度统计https://tongji.baidu.com 新增网站，添加完毕后到代码获取拿到统计脚本,找到https://hm.baidu.com/hm.js?后面的key值
  2. 拿到key后，去到Hexo根目录下的themes\next\，打开 _config.yml 文件，找到baidu_analytics，在后面加上key值
  3. 然后重新编译，打包上传就可以了 hexo d -g
  4. 测试，去到百度统计，找到代码安装检查，就可以测试是否添加上去了

    LeanCloud阅读次数统计

  • 首先，前往Leancloud官网进行注册，并登陆。

  • 点击创建应用,输入应用的名称(随便起都行), 选择开发版, 点击创建

  • 创建成功后点击右上角的设置小齿轮,点击存储-创建 Class

  • Class名称设置为Counter(必须为Counter)-选择无限制-点击创建

  • 创建后点击设置-点击应用Key-获取App ID和App Key

  • 将获取到的App ID和App Key设置到Next主题配置文件 _config.yml,配置文件已存在这个配置, 只要把id和key加上去就好了

  • 点击安全中心, 填写自己博客对应的域名（注意协议、域名和端口号需严格一致）

  • 点击云引擎-点击部署-点击在线编辑

  • 点击创建函数-选择 Hook , AV.Cloud.选择 beforeSave ,类选择Counter-添加代码

  • 点击保存，点击部署

  • 打开Next主题配置文件 _config.yml, 将leancloud_visitors下的security设置为true（如没有则新增）

  • 打开命令行cmd并进入站点文件夹，npm install hexo-leancloud-counter-security --save

  • 设置hexo lc-counter register <<username>> <<password>>，将<< username >>和<< password >>替换为你自己的用户名和密码（不必与leancloud的账号）相同。

  • 打开站点配置文件 _config.yml，将<< username >>和<< password >>替换为你刚刚设置的用户名和密码

  • 返回Leancloud控制台的应用内, 检查_User表中是否出现一条记录
  • 进入Counter表, 打开权限设置，修改add和create为指定用户，修改delete为指定空用户
